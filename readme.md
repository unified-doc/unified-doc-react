## Development
```sh
# clean package (rm dist + node_modules)
npm run clean

# lint package with xo + prettier + tsc
npm run lint

# watch/rebuild package with microbundle
npm run dev

# test package with jest (make sure to run the 'dev' script)
npm run test

# build package with microbundle
npm run build
```
